# 情報I 学習ポータルサイト

実教出版の教科書「最新情報I」の学習体系に基づいた、高校生向けの学習支援ポータルサイトです。

## 🚀 公開URL
[https://n-ryo-jp.github.io/joho1-portal/]
## 📘 このサイトで学べること
- **第1章：情報社会の問題解決** (情報デザイン、知的財産権)
- **第2章：コミュニケーションと情報デザイン** (2進法、デジタル化)
- **第3章：コンピュータとネットワーク** (Pythonプログラミング、通信手順)
- **第4章：データ活用** (統計、シミュレーション)

## 🛠️ 使用技術
- HTML5 / CSS3
- Tailwind CSS (デザインフレームワーク)
- Phosphor Icons (アイコン)
